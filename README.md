# Notes CRUD API

A standalone Node.js and Express backend that performs full CRUD operations on a MongoDB database. This project was built for the CRUD Operations with Express assignment.

## Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.
- A MongoDB connection URI (e.g., via MongoDB Atlas).
